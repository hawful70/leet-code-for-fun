# LeetCode for Fun

A personal collection of LeetCode solutions written in **Go** and **Python**, created just for the fun of it.

## Overview

- **175+ problems** solved and counting
- Every problem is implemented in both **Go** and **Python**
- Solutions live in the `daily/` folder, named by problem number and title

## Topics Covered

| Topic | Example Problems |
|---|---|
| Arrays & Two Pointers | Two Sum, 3Sum, 4Sum, Trapping Rain Water, Container With Most Water |
| Sliding Window | Longest Substring Without Repeating Characters, Minimum Window Substring, Permutation in String |
| Binary Search | Search in Rotated Sorted Array, Koko Eating Bananas, Median of Two Sorted Arrays |
| Linked Lists | Reverse Linked List, Merge K Sorted Lists, LRU Cache, LFU Cache |
| Stacks & Queues | Valid Parentheses, Min Stack, Evaluate Reverse Polish Notation |
| Trees | Binary Tree Inorder/Preorder/Postorder Traversal, Same Tree, Level Order Traversal |
| Hash Maps & Sets | Group Anagrams, Contains Duplicate, Top K Frequent Elements |
| Prefix Sum | Range Sum Query, Subarray Sum Equals K |
| Sorting | Sort Colors, Sort Array, Squares of a Sorted Array |
| Greedy | Assign Cookies, Boats to Save People, Bag of Tokens |

## Running Solutions

**Go**
```bash
cd daily
go run <problem-file>.go
```

**Python**
```bash
cd daily
python3 <problem-file>.py
```
